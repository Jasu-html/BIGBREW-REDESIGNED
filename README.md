This repository contains the ongoing project for the **BigBrew Promotional Website Redesign**, created as part of my final project for the **CIS-211** course. The goal is to redesign the **BigBrew** website with a modern, user-friendly interface, improving both its aesthetics and functionality.

At this stage, the website is still under development. This README serves to provide an overview of the project, its current state, and the technologies being used.
